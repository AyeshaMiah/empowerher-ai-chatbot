# EmpowerHer AI Chatbot 💜

An AI-powered chatbot designed to support and empower women by providing guidance, resources, and helpful conversations.

## Features

- AI chatbot using Google Gemini API
- Natural language conversations
- User-friendly interface
- Fast response generation

## Technologies Used

- Python
- Streamlit
- Google Gemini API

## Installation

Clone the repository:

git clone https://github.com/rasheeda6786/empowerher-ai-chatbot.git

Install dependencies:

pip install -r requirements.txt

Run application:

streamlit run app.py
